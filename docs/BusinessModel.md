## 1. 비즈니스 모델 캔버스

### 핵심 파트너 (Key Partners)
- 카카오 (알림톡 API, 소셜 로그인, 페이)
- 지역 학원 협회 (온보딩 파트너)
- 문화체육관광부, 교육부 (평생교육바우처 등 바우처 연동)
- PG사 (결제: 토스페이먼츠, KG이니시스)
- 지역 화폐 운영사 (서울페이, 경기지역화폐)
- 셔틀 버스 운영 업체 (데이터 연동)

### 핵심 활동 (Key Activities)
- 공급자(강사/학원) 온보딩 및 품질 관리
- 위치 기반 매칭 알고리즘 개발
- 셔틀 연계 시간표 엔진 개발
- 인증 리뷰 시스템 운영 (영수증 검증)
- 커뮤니티 운영 (어뷰징 방지)
- 카카오 알림톡 연동 유지
- 마케팅 (맘카페 제휴, 강사 네트워크)

### 핵심 자원 (Key Resources)
- 강사/학원 DB (공급)
- 인증 리뷰 데이터 (차별화)
- 셔틀 노선 데이터
- 지역별 학부모 커뮤니티 (네트워크 효과)
- 개발팀 (풀스택 + iOS/Android)

### 가치 제안 (Value Propositions)

| 타겟 | 가치 제안 |
|---|---|
| 학부모 | 모든 학원 일정·결제·공지를 하나의 앱에서. 셔틀 연계까지. |
| 강사/원장 | 행정 자동화(공지·결제·셔틀)로 수업에만 집중. 신규 수강생 유입. |
| 성인 수강생 | 내 빈 시간에 맞는 진짜 후기 기반 취미 수업 탐색. |

### 고객 관계 (Customer Relationships)
- 학부모: 자녀 성장 콘텐츠(리포트·사진)로 감성적 유대
- 강사: 행정 효율 + 비즈니스 성장 파트너로 포지셔닝
- 성인: 성장 포트폴리오 + 커뮤니티로 커뮤니티 소속감

### 채널 (Channels)
- iOS / Android 앱
- PWA (Progressive Web App) — 설치 없이 사용
- 카카오 알림톡 (비앱 사용자 접점)
- 강사 초대 링크 (바이럴: 강사 → 학부모)
- 맘카페 파트너십 (정보 허브 제휴)

### 고객 세그먼트 (Customer Segments)
1. **초·중·고 학부모** (메인, B2C)
2. **성인 취미 수강생** (성장, B2C)
3. **강사/소규모 학원** (공급자, B2B2C)
4. **학원 프랜차이즈** (미래, B2B Enterprise)

### 비용 구조 (Cost Structure)
- 인건비 (개발·디자인·영업·CS): ~60%
- 서버 인프라 (AWS): ~10%
- 마케팅 (CAC): ~20%
- 카카오 알림톡 발송 비용: ~5%
- 법무·회계: ~5%

## 2. 수익 모델 상세

### 2.1 거래 수수료 (핵심 수익) — Phase 1

| 대상 | 수수료 | 비고 |
|---|---|---|
| 신규 등록 결제 | 5% | 강사에게 부과 |
| 월납 자동 결제 | 3% | 장기 수강생 우대 |
| 수업 개설 펀딩 성공 시 | 5% | 미달 시 무료 |

**예시 시뮬레이션**

| 지표 | 값 |
|---|---|
| 월 거래액 | 5억 원 |
| 평균 수수료율 | 4% |
| **월 수수료 수익** | **2,000만 원** |

### 2.2 강사 구독 플랜 (SaaS) — Phase 2

| 플랜 | 월 요금 | 기능 |
|---|---|---|
| 무료 (Starter) | 0원 | 프로필 등록, 최대 수강생 10명 |
| 스탠다드 | 29,000원 | 무제한 수강생, 공지 자동화, 결제 관리 |
| 프로 | 59,000원 | 스탠다드 + 강사 광고 노출, 리포트 고급 템플릿 |
| 엔터프라이즈 | 협의 | 학원 프랜차이즈, 다지점 관리, API 연동 |

**수익 시뮬레이션 (300명 강사 기준)**

| 플랜 | 비율 | 인원 | 월 수익 |
|---|---|---|---|
| 무료 | 50% | 150 | 0 |
| 스탠다드 | 35% | 105 | 3,045,000 |
| 프로 | 15% | 45 | 2,655,000 |
| **합계** | | | **5,700,000원** |

### 2.3 광고 & 프리미엄 노출 — Phase 3

| 상품 | 가격 | 설명 |
|---|---|---|
| 지역 탑 배너 | 월 50만 원/구 | 지역 검색 결과 상단 고정 |
| 추천 강사 뱃지 | 월 20만 원 | "에듀픽 추천" 표시 |
| 신규 수강생 광고 | CPC 500원 | 클릭당 과금 |

### 2.4 데이터 인사이트 리포트 — Phase 3 (B2B)

- 지역별 학원 트렌드 리포트 (학원 컨설팅사, 부동산, 교육청 대상)
- 연간 구독 500만 원 ~ 2,000만 원

## 3. 수익 성장 로드맵

```
월 수익 목표

Phase 1 (M1~M6): 거래 수수료 중심
  M1~M2: 베타, 수수료 무료 → 0원
  M3~M4: 수수료 3% 적용 → ~500만 원/월
  M5~M6: 강사 300명 목표 → ~2,000만 원/월

Phase 2 (M7~M12): SaaS 추가
  강사 구독 + 수수료 → ~5,000만 원/월

Phase 3 (M13~M24): 광고 + 데이터
  전체 수익 → ~1억 원/월 목표 (BEP 달성)
```

## 4. Unit Economics

### 강사 LTV (Lifetime Value)
```
ARPU (강사, 스탠다드 기준): 29,000원/월
평균 유지 기간: 24개월
LTV = 29,000 × 24 = 696,000원

+ 거래 수수료 (월 수강생 30명 × 15만 원 × 4%) = 18만 원/월
거래 수수료 LTV = 18만 × 24 = 432만 원

강사 1인 총 LTV ≈ 500만 원
```

### 학부모 LTV
```
ARPU (광고 없는 구조에서 간접 기여):
  수수료 기반 → 강사 비용에 내재화
  향후 프리미엄 기능 (패밀리 구독 9,900원/월)

패밀리 구독 LTV = 9,900 × 18 = 178,200원
```

### CAC (고객 획득 비용) 목표
- 강사: 5만 원 이하 (오프라인 영업 포함)
- 학부모: 1만 원 이하 (강사 초대 바이럴 활용)

## 5. 핵심 성장 가설 (Growth Flywheel)

```
강사 온보딩
     ↓
학부모 초대 (강사가 에듀픽 링크 발송)
     ↓
학부모 일정 관리 + 리뷰 작성
     ↓
신뢰도 높은 리뷰 데이터 축적
     ↓
신규 학부모/수강생 유입 (검색, 입소문)
     ↓
강사 수익 증가 → 추가 강사 온보딩
     ↑________________________↓
         (플라이휠 완성)
```

**핵심**: 강사가 에듀픽을 학부모에게 소개함으로써 **B2B2C 바이럴**이 발생한다. CAC를 최소화하는 핵심 구조.
