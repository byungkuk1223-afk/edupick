# 에듀픽(EduPick) 경쟁사 분석 (Competitive Analysis)

**작성일**: 2026-03-11

---

## 1. 경쟁 구도 개요

에듀픽은 단일 경쟁자가 없는 **복합 시장**에 진입한다. 현재 시장은 기능별로 분산되어 있어, 에듀픽이 통합 솔루션으로 포지셔닝할 여지가 크다.

```
       정보탐색
          ↑
  맘카페  │  네이버 지도
  블로그  │  클래스101
          │
결제/관리 ◄────────────────► 일정관리
 클래스패스│                  에브리타임
  알라딘클 │                  구글 캘린더
          │
          ↓
       커뮤니티
         밴드
         카톡
```

**에듀픽의 포지션**: 위 4개 축을 하나로 통합하는 중심부

## 2. 직접 경쟁사

### 2.1 클래스101 (Class101)

| 항목 | 내용 |
|---|---|
| **유형** | 성인 취미 온라인 클래스 플랫폼 |
| **강점** | 콘텐츠 품질 보증, 영상 강의 특화, 브랜드 인지도 |
| **약점** | 오프라인 로컬 수업 없음, 실시간 상호작용 부재 |
| **에듀픽 차별점** | 오프라인 로컬 강사 + 실시간 시간표 매칭 |

### 2.2 레슨올 (LessonAll)

| 항목 | 내용 |
|---|---|
| **유형** | 개인 레슨 강사 매칭 플랫폼 |
| **강점** | 강사 DB, 과목 다양성 |
| **약점** | 학부모 관리 기능 없음, 결제/공지 미지원, 커뮤니티 부재 |
| **에듀픽 차별점** | 학원 관리(공지·결제·셔틀) + 커뮤니티 통합 |

### 2.3 하이클래스 (Hiclass)

| 항목 | 내용 |
|---|---|
| **유형** | 학원 관리 앱 (강사→학부모 소통) |
| **강점** | 학원 관리 특화, 출결 관리 |
| **약점** | 수강생 탐색 기능 없음, UI 구식, 성인 시장 미지원 |
| **에듀픽 차별점** | 탐색(검색) + 관리를 하나로, 성인 취미 시장 포함 |

### 2.4 클래스패스 (ClassPass, 해외)

| 항목 | 내용 |
|---|---|
| **유형** | 피트니스/웰니스 구독형 클래스 예약 |
| **강점** | 다양한 헬스 클래스, 구독 모델로 높은 LTV |
| **약점** | 국내 시장 철수, 학원·교육 시장 미포함 |
| **에듀픽 차별점** | 교육(학생) + 취미(성인) 통합, 한국 로컬 특화 |

## 3. 간접 경쟁사 (대체재)

### 3.1 카카오톡 단체방
- **현재 역할**: 학원-학부모 간 모든 커뮤니케이션
- **문제**: 공지 휘발, 프라이버시, 행정 비효율
- **에듀픽 전략**: 카카오 알림톡 연동으로 "카톡을 버리게 하지 않고 카톡을 더 잘 쓰게 해주는 것"처럼 초기 진입

### 3.2 네이버 카페 (맘카페)
- **현재 역할**: 학원 정보 탐색, 리뷰 공유
- **문제**: 광고성 글 혼재, 폐쇄적 접근, 검색 불편
- **에듀픽 전략**: 영수증 인증 리뷰로 신뢰도 높은 정보 허브 제공

### 3.3 네이버 지도 / 카카오 지도
- **현재 역할**: 학원 위치 검색
- **문제**: 수업 시간표, 셔틀, 리뷰 품질 부족
- **에듀픽 전략**: 학원 전문 지도 + 셔틀 시각화

### 3.4 구글 캘린더 / 에브리타임
- **현재 역할**: 개인 일정 관리
- **문제**: 교육 특화 기능 없음 (셔틀 연계, 준비물 알림 등)
- **에듀픽 전략**: 교육 도메인 특화 캘린더

## 5. 포지셔닝 전략

### 핵심 포지션
> "카톡의 혼란을 정리하고, 맘카페의 정보를 믿을 수 있게 만들고, 에브리타임처럼 내 시간에 맞는 수업을 찾아주는 **학원계의 All-in-One 플랫폼**"

### 초기 진입 전략 (Beachhead)
1. **강남 3구 (강남·서초·송파) 초등 학부모 집중 공략** — 학원 밀집, 디지털 친화, 지불 의사 높음
2. **발레·피아노·미술 강사 우선 온보딩** — 기존 카톡 의존도 높고 SNS 활용 니즈 있음
3. **성인 취미 시장은 Phase 2** — 초기 트랙션 후 확장

### 차별화 메시지
- 학부모: *"더 이상 카톡 공지 뒤지지 마세요"*
- 강사: *"수업에만 집중하세요. 행정은 에듀픽이"*
- 성인 수강생: *"내 빈 시간에 딱 맞는 취미를"*
