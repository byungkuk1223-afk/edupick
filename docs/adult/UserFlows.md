## 1. 성인 온보딩 플로우

```
[앱 다운로드 / PWA 접속]
        ↓
[카카오 로그인]
        ↓
[역할 선택]
  "취미를 배우고 싶어요 (성인 수강생)"
        ↓
[관심 카테고리 선택] (복수)
  발레 / 피아노 / 수채화 / 도예 / 기타 / 요가 / ...
        ↓
[위치 권한 허용]
  → 허용: 현재 위치 기반 수업 탐색
  → 거부: 거주지/직장 주소 수동 입력
        ↓
[고정 일정 입력 (선택)]
  "더 정확한 추천을 위해 내 바쁜 시간대를 알려주세요"
  → 건너뛰기 가능 (나중에 입력)
        ↓
[첫 추천 수업 3개 표시]
  "이 시간에 이 근처에서 들을 수 있어요"
        ↓
[메인 홈으로 이동]
```

## 2. 스마트 시간표 빌더 플로우

```
[시간표 탭 진입]
        ↓
[내 고정 일정 입력 화면]
  ┌─────────────────────────────────┐
  │ + 일정 추가                      │
  │ ┌───────────────────────────┐   │
  │ │ 직장       월~금 09:00~19:30│  │
  │ │ 개인 약속   토 13:00~15:00 │   │
  │ └───────────────────────────┘   │
  └─────────────────────────────────┘
        ↓
[분석 중... (1~2초)]
  "토요일 오전 10:00~12:30,
   평일 19:30 이후에 수업을 들을 수 있어요"
        ↓
[관심 카테고리 + 빈 시간 + 위치 기반 매칭]
        ↓
[추천 수업 카드 리스트]
  ┌─────────────────────────────────┐
  │ 🩰 성인 발레 초급반              │
  │ 이지현 강사 | 역삼역 도보 8분    │
  │ 토요일 10:30~11:30 | ★4.8       │
  │ "왜 추천했나요?" 클릭            │
  │ → "토요일 빈 시간 + 내 관심사"  │
  └─────────────────────────────────┘
        ↓
[수업 상세 → 인증 리뷰 확인 → 신청]
```

## 3. 수업 탐색 → 신청 플로우

```
[탐색 탭]
  ├─ 지도 뷰 (카테고리별 컬러 핀)
  └─ 리스트 뷰 (기본 정렬: 거리순)
        ↓
[필터 설정]
  - 카테고리: 발레
  - 시간대: 19시 이후 또는 주말
  - 수업 유형: 정규반 / 원데이 / 펀딩 수업
  - 가격: ~15만 원/월
        ↓
[학원/강사 카드 클릭]
        ↓
[상세 페이지]
  ├─ 강사 소개 (경력, 자격증, 스타일)
  ├─ 수업 시간표
  ├─ 인증 리뷰 (항목별 평점)
  │    "강사가 손 모양 꼼꼼히 잡아주나요?" Q&A
  ├─ 사진
  └─ 수강료 안내
        ↓
[신청하기]
  - 수업 시간대 선택
  - 결제 유형 선택 (월납 / 횟수권 / 원데이)
  - 결제 수단 선택
  - 바우처 코드 입력 (선택)
        ↓
[결제 완료]
  - 인보이스 발급
  - 강사에게 신규 수강생 알림
  - 캘린더 자동 등록
```

## 4. 수업 개설 펀딩 참여 플로우

```
[탐색 탭 → "펀딩 수업" 필터 ON]
        ↓
[펀딩 수업 카드]
  ┌─────────────────────────────────────┐
  │ 🩰 토요일 오전 성인 발레 초급반      │
  │ 이지현 강사 | 역삼역 도보 8분        │
  │ ████████░░  현재 3명 / 최소 3명      │
  │ 마감까지 D-7                         │
  │ "1명만 더 모이면 확정!"              │
  └─────────────────────────────────────┘
        ↓
[펀딩 상세 페이지]
  - 수업 소개, 강사 프로필
  - 참여자 수 실시간 현황
  - 예상 수업 일정 미리보기
        ↓
[사전 신청]
  - 이름 / 연락처 자동 입력
  - 한마디 (선택): "취미로 배우고 싶어요"
  → 예약금 없음, 확정 시 결제 안내
        ↓
━━━━━━━━━━━━━━━━━━━━━━━
[케이스 A: 최소 인원 달성]
  → 알림: "🎉 수업 확정! 결제를 완료해 주세요"
  → 결제 → 캘린더 등록

[케이스 B: 마감 미달]
  → 알림: "아쉽게도 이번 수업은 열리지 않았어요"
  → "비슷한 수업 3개 추천" 자동 표시
━━━━━━━━━━━━━━━━━━━━━━━
```

## 5. 인증 리뷰 작성 플로우

```
[수강 등록 후 30일 경과]
        ↓
[푸시 알림]
  "발레 수업 후기를 남겨주세요!
   실제 수강생만 쓸 수 있어요 💬"
        ↓
[리뷰 작성 화면]
  1. 별점 선택 (1~5)
  2. 항목별 평점
     - 강사 전문성
     - 커리큘럼 구성
     - 시설/환경
     - 수업 분위기
  3. 자유 후기 텍스트
  4. 특정 질문 답변 (선택)
     "강사가 개인 자세 교정을 해주나요?" → Yes / No / 상세 답변
        ↓
[결제 인증]
  → 앱 내 결제: 자동 인증 (버튼 없음)
  → 외부 결제: 영수증 이미지 업로드
     → OCR 자동 분석 → 학원명/금액 매칭
        ↓
[인증 완료 → 리뷰 등록]
  → 내 리뷰에 "인증된 수강생" 뱃지
  → 강사에게 새 리뷰 알림
  → 다음 달 수강료 500원 할인 쿠폰 지급
```

## 7. 성인 모드 화면 구조

```
메인 탭바
├── 🏠 홈
│   ├── 이번 주 수업 카드
│   ├── 잔여 횟수권
│   └── "지금 등록 가능한 수업" 배너
│
├── 🔍 탐색
│   ├── 지도 뷰
│   ├── 리스트 뷰
│   │   └── 필터: 시간대 / 카테고리 / 가격 / 수업유형
│   └── 펀딩 수업 탭
│       ├── 진행 중 펀딩 카드
│       └── 마감 임박 우선 표시
│
├── 📅 내 시간표
│   ├── 고정 일정 입력/편집
│   ├── 빈 시간대 시각화 (블록 형태)
│   └── 추천 수업 매칭 카드
│
├── 🌱 성장 기록
│   ├── 과목 탭 전환
│   ├── 타임라인 (날짜순)
│   ├── 미디어 갤러리
│   └── 공유 버튼
│
└── 👤 마이페이지
    ├── 결제 내역 / 횟수권 현황
    ├── 내 리뷰 관리
    ├── 알림 설정
    └── 바우처 관리
```
