## Persona A1 — "취미를 찾고 싶은 번아웃 직장인" 박지호 (32세)

### 기본 정보
- **직업**: IT 스타트업 백엔드 개발자
- **거주지**: 서울 마포구 (합정역 근처)
- **가족**: 1인 가구
- **성격**: 내향적, 유튜브 알고리즘에 취미 콘텐츠(발레·기타·수채화) 자주 등장

### 라이프스타일
```
평일: 출근 09:30 / 퇴근 20:00~21:00 (야근 잦음)
주말: 토 오전은 대부분 자유, 일 오후 여유 시간
SNS: 인스타 팔로잉의 30%가 취미 계정
최근 고민: "운동 말고 뭔가 창의적인 걸 배우고 싶다"
```

### 페인 포인트
- "강남 쪽 성인 발레 학원 검색하면 광고 블로그밖에 없어."
- "퇴근이 8시인데 맞는 수업 시간대가 없거나 정보 자체가 없어."
- "원데이 클래스 혼자 신청하기 뭔가 어색해. 3명 이상 모이면 개설 같은 거 없나?"
- "3개월 배웠는데 내가 얼마나 늘었는지 모르겠어. 동기가 안 생겨."

### 에듀픽 사용 시나리오
1. 퇴근 시간(20:00 이후) + 합정역 근처 → 성인 발레 클래스 자동 추천
2. "수강생만 쓴 진짜 후기" — 인증 리뷰 확인 후 결정
3. "토요일 오전 성인 발레 3명 모이면 개설" 펀딩에 사전 신청
4. 6개월 후 영상 포트폴리오 → 친구에게 공유

### 인용구
> "유튜브에서 발레 영상 보면서 나도 해보고 싶다 싶었는데,
> 어디서 어떻게 시작해야 하는지 막막했어. 내 퇴근 시간에
> 맞는 수업을 딱 찾아준다면 바로 등록할 텐데."

### 기본 정보
- **직업**: 중학교 국어 교사 (주 5일)
- **거주지**: 경기 성남시 분당구
- **가족**: 남편, 초등 2학년 아들
- **과거 취미**: 20대 초반까지 피아노 전공 지망 → 포기

### 라이프스타일
```
평일: 학교 08:00~16:30, 아이 학원 관리, 저녁 시간 여유 없음
주말 토: 오전 자유 (남편이 아이 담당)
최근 변화: 육아에만 매몰된 지난 7년에 대한 아쉬움
```

### 페인 포인트
- "피아노 다시 시작하고 싶은데 성인반이 있는 학원이 어딘지 몰라."
- "맘카페에서 물어보면 '저 피아노 선생님 어떠냐'는 질문이 어색해."
- "수업 다니다가 실력이 느는 게 눈에 보여야 지속하게 되더라고."

### 에듀픽 사용 시나리오
1. "성인 피아노" + "토요일 오전" + "분당" 필터 → 3개 수업 추천
2. "7년 수강생 후기: 처음부터 다시 시작해도 잘 가르쳐줌" 인증 리뷰 확인
3. 수강 시작 → 매달 강사 피드백 + 연주 영상 타임라인 기록
4. 6개월 후 처음 연주 영상 vs 최근 영상 비교 → 동기 유지

### 인용구
> "아이 학원비는 매달 체계적으로 관리하면서
> 내 취미는 '언젠가 시작해야지'만 10년째야.
> 내 빈 시간에 맞는 수업 찾아주고 내 성장도 기록해준다면
> 진짜 시작할 수 있을 것 같아."

### 기본 정보

- **거주지**: 서울 성동구 (성수동)
- **가족**: 1인 가구
- **소비 패턴**: 월 5~10만 원 이상 원데이 클래스 지출

### 라이프스타일
```
주말: 인스타그램에서 발견한 클래스 도전
관심사: 도예, 드로잉, 플라워, 요리, 캔들
고민: "매번 새 곳 찾는 게 귀찮아. 한 곳에서 볼 수 없나?"
```

### 페인 포인트
- "원데이 클래스 정보가 인스타, 크몽, 네이버에 흩어져 있어."
- "예약했는데 갑자기 취소된 적이 있어서 믿을 수 있는 곳이 필요해."
- "다녀온 곳 기록을 내가 직접 해야 해. 자동으로 모아줬으면."

### 에듀픽 사용 시나리오
1. "이번 주말 성수동 원데이 클래스" → 토/일 가능한 클래스 리스트
2. "영수증 인증 후기만 보기" → 광고 없는 실후기 필터
3. 수강 완료 → 성장 타임라인 자동 추가 (날짜, 사진, 강사 코멘트)
4. 6개월 누적 → "나 이런 거 다 해봤네" 포트폴리오 완성

### 에듀픽 사용 시나리오

3. 에듀픽 강사 프로필 → 인증 리뷰 누적 → 신규 유입
