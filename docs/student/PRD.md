## 1. 개요

### 제품 비전

> "더 이상 카톡 공지 뒤지지 마세요. 에듀픽만 열면 오늘 아이의 동선, 준비물, 셔틀 위치가 한눈에 보입니다."

학원 정보 탐색부터 일정 관리, 셔틀 연계, 아이 성장 기록까지 — 학부모의 **'스케줄링 헬(Scheduling Hell)'** 을 해소하는 학원 통합 관리 플랫폼.

### 핵심 문제

| 문제 | 현재 해결 방법 | 한계 |
|---|---|---|
| 학원 공지 휘발 | 카톡 단체방 스크롤 | 공지 찾는 데 10분, 놓치면 큰일 |
| 아이 사진 프라이버시 | 단체방 공유 | 모르는 사람에게 노출 |
| 멀티 학원 시간표 맞추기 | 엑셀·메모 | A학원 끝나고 B학원 셔틀 탈 수 있는지 모름 |
| 신뢰할 수 있는 학원 정보 | 맘카페 탐색 | 광고 혼재, 폐쇄적 접근 |
| 매달 학원비 납부 확인 | 문자·카톡 개별 확인 | 누락, 강사 행정 부담 |

## 2. 타겟 세그먼트 상세

### 초등학생 학부모 (Care 중심)
- **핵심 니즈**: 안심과 케어
- 셔틀 위치·도착 알림, 귀가 확인
- 오늘 수업 사진, 준비물 알림
- "우리 아이 잘 있나?" 실시간 안심 정보

### 중·고등학생 학부모 (Result 중심)
- **핵심 니즈**: 학습 효율
- 내신 대비 진도 리포트
- 시험 기간 특강 일정 관리
- 보강 예약, 결석 처리
- 학원 성과 비교 (어디서 성적이 올랐나)

## 4. 연령별 UX 분기

### 초등 학부모 뷰
```
홈 화면 카드 구성
├── 📍 셔틀 현황 ("잠시 후 5분 뒤 도착")
├── 🍱 오늘 활동 사진 (새 리포트 있음)
├── 📋 내일 준비물 ("실내용 운동화")
└── 💳 이번 달 납부 현황
```

### 중·고등 학부모 뷰
```
홈 화면 카드 구성
├── 📅 이번 주 보강/특강 일정
├── 📊 최근 진도 리포트 (국어 학원)
├── ⚠️ 시험까지 D-14, 특강 3회 남음
└── 💳 다음 달 납부 예정 XX만 원
```

## 6. 리스크

| 리스크 | 대응 |
|---|---|
| 앱 설치 거부감 | PWA + 카카오 알림톡으로 설치 없이 핵심 기능 이용 가능 |
| 아동 사진 보안 우려 | S3 private bucket + pre-signed URL, 본인 아이 사진만 접근 가능 |
| 학원별 공지 형식 통일 어려움 | 강사용 공지 템플릿 제공으로 표준화 유도 |
| 셔틀 데이터 정확도 | 강사가 직접 노선 등록, 변경 시 즉시 알림 의무화 |
| 중·고등 학부모의 낮은 앱 신뢰 | 내신 리포트 실효성 증명이 핵심 — 성적 향상 사례 마케팅 |
