## 1. 학부모 온보딩 플로우

```
[앱 다운로드 OR 강사 초대 링크 클릭]
        ↓
[카카오 로그인]
        ↓
[역할 선택]
  "아이 학원을 관리하고 싶어요 (학부모)"
        ↓
[자녀 정보 입력]
  ┌──────────────────────────────┐
  │ + 첫째 아이 추가              │
  │  이름: 민준                   │
  │  학교: 역삼초등학교 4학년     │
  │                              │
  │ + 둘째 아이 추가              │
  │  이름: 수아                   │
  │  학교: 역삼초등학교 1학년     │
  └──────────────────────────────┘
        ↓
[위치 권한 허용]
        ↓
[학원 연결 방법 선택]
  A. 강사 초대 코드 입력 → 즉시 학원 연결
  B. 직접 탐색해서 등록
        ↓
[메인 대시보드 진입]
```

## 2. 학원 탐색 → 등록 플로우 (학부모)

```
[탐색 탭]
        ↓
[검색 조건 설정]
  - 자녀 선택 (민준 / 수아)
  - 학교급: 초등
  - 과목: 수학
  - 위치: 현재 위치 반경 2km
  - 셔틀 있는 학원만 보기: ON
        ↓
[지도 / 리스트 뷰]
  → 학원 카드에 셔틀 정류장 아이콘 표시
        ↓
[학원 상세 페이지]
  ├─ 기본 정보 (위치, 수강료, 강사 소개)
  ├─ 셔틀 노선 지도
  │    "우리 아파트(역삼 래미안 1단지) 정류장 있음 ✓"
  ├─ 수업 시간표 (남은 자리 표시)
  ├─ 인증 리뷰 (초등 항목: 케어·안전·시설·셔틀)
  └─ [셔틀 연계 확인] 버튼
        ↓
[셔틀 연계 확인] 클릭 시
  "기존 수업과 연계 가능 여부 확인"
  → 영어 학원 17:00 종료 → 셔틀 17:10 출발 →
     역삼 래미안 1단지 17:30 하차 →
     도보 3분 → 수학 학원 셔틀 17:33 탑승 →
     수학 학원 18:00 도착 ✓
  → "연계 가능합니다! (여유 시간 27분)"
        ↓
[수강 신청]
  - 자녀 선택
  - 수업 시간대 선택
  - 결제 방법 (카드 / 카카오페이 / 계좌이체)
  - 바우처 코드 입력 (선택)
        ↓
[등록 완료]
  - 캘린더 자동 추가 (자녀별 색상 구분)
  - 강사에게 신규 등록 알림
  - 셔틀 알림 자동 ON
```

## 3. 셔틀 알림 수신 플로우 (학부모 측)

```
[강사가 앱에서 "셔틀 출발" 탭]
        ↓
[에듀픽 서버: 해당 수업 수강생 학부모 식별]
        ↓
[알림 동시 발송]
  → 앱 설치 학부모: 푸시 알림
  → 앱 미설치 학부모: 카카오 알림톡
        ↓
[알림 내용]
  "🚐 잠시 후 5분 뒤
   역삼 발레 아카데미 셔틀이
   [역삼 래미안 1단지]에 도착합니다."
        ↓
[알림 탭 → 확인 표시]

━━━━━━━━━━━━━━━━━━━━━
[수업 종료 귀가 셔틀 알림]
  "✅ 오늘 발레 수업 종료.
   귀가 셔틀 출발 예정 17:55
   [역삼 래미안 1단지] 도착 18:10"
━━━━━━━━━━━━━━━━━━━━━
```

## 4. 프라이빗 아이 리포트 플로우

```
[강사: 수업 후 리포트 작성]
  - 자녀 선택 (민준 / 수아 개별)
  - 사진 업로드 (수업 활동 사진)
  - 코멘트 입력:
    "오늘 팔 자세 많이 늘었어요! 다음 주는 발 포지션 집중 예정"
        ↓
[학부모 수아 엄마에게만 알림]
  "📸 오늘 수아의 발레 수업 사진이 등록됐어요"
        ↓
[학부모 앱 → 리포트 탭 → 수아 선택]
  ┌────────────────────────────────┐
  │ 수아의 발레 수업 기록           │
  │                                │
  │ 3월 10일 월요일                 │
  │ 📸 [수업 사진]                  │
  │ 💬 "오늘 팔 자세 많이 늘었어요" │
  │                                │
  │ 3월 3일 월요일                  │
  │ 📸 [수업 사진]                  │
  │ 💬 "기초 포르 드 브라 시작"     │
  └────────────────────────────────┘

※ 수아 사진은 수아 엄마만 볼 수 있음
※ 단체방에 공개되지 않음
```

## 5. 스마트 시간표 빌더 (학원 추가) 플로우

```
[캘린더 탭 → "학원 추가 가능 여부 확인"]
        ↓
[현재 민준 일정 분석]
  월: 영어 16:30~18:00 (셔틀 귀가 18:20)
  수: 수학 17:00~19:00 (셔틀 귀가 19:20)
  금: 피아노 17:30~18:30
        ↓
[빈 시간대 분석 결과]
  "화요일, 목요일 오후에 학원을 추가할 수 있어요.
   셔틀 이동 시간을 고려하면
   화: 16:00~19:00 가능
   목: 16:00~19:00 가능"
        ↓
[추천 수업 표시]
  - 화요일 17:00 농구 (도보 10분) ← 셔틀 없음
  - 목요일 16:30 미술 학원 (셔틀 있음) ← 추천
        ↓
[목요일 미술 학원 선택 → 상세 확인 → 등록]
```

## 6. 결석/보강 신청 플로우

```
[학부모 앱 → 해당 수업 카드 → "이번 주 결석"]
        ↓
[결석 사유 입력 (선택)]
  가정 사정 / 아픔 / 기타
        ↓
[강사에게 알림]
  "민준 어머니: 이번 주 화요일 수업 결석 예정"
        ↓
[강사 → 보강 가능 일정 제안]
  "다음 주 토요일 10:00에 보강 가능합니다"
        ↓
[학부모 → 확인 또는 다른 시간 요청]
        ↓
[보강 일정 캘린더 자동 추가]
```

## 7. 인증 리뷰 작성 플로우 (학부모)

```
[수강 등록 후 30일 경과]
        ↓
[푸시 알림]
  "민준이 영어 학원 후기를 남겨주세요.
   영수증 인증 후기만 표시됩니다 🔒"
        ↓
[리뷰 작성]
  별점 + 항목별 평점 (초등용)
  - 아이 케어 수준
  - 시설 안전
  - 셔틀 정확도
  - 수업 내용
  자유 후기 텍스트
        ↓
[인증: 앱 내 결제 → 자동 인증]
        ↓
[등록 완료]
  → "인증된 학부모 후기" 뱃지
```

## 8. 화면 구조 (학부모 모드)

```
메인 탭바
├── 🏠 홈 (대시보드)
│   ├── 오늘 [민준] [수아] 탭 전환
│   ├── 셔틀 알림 카드 (도착 임박 시 강조)
│   ├── 새 리포트 알림
│   ├── 내일 준비물 미리 보기
│   └── 이번 달 결제 현황 요약
│
├── 🔍 탐색
│   ├── 지도 뷰 (학원 핀)
│   ├── 리스트 뷰 (거리순)
│   ├── 필터: 자녀 학교급 / 과목 / 셔틀 / 바우처
│   └── 학원 상세
│       ├── 시간표 + 잔여 자리
│       ├── 셔틀 노선 지도
│       ├── 연계 가능 여부 확인
│       └── 인증 리뷰
│
├── 📅 캘린더
│   ├── 월간 뷰 (자녀별 색상)
│   ├── 일간 뷰 (시간대별 수업 + 셔틀)
│   ├── 학원 추가 가능 여부 확인
│   └── 공휴일 / 시험 일정 표시
│
├── 🔔 알림
│   ├── 셔틀 도착 알림
│   ├── 공지 (준비물, 일정 변경)
│   ├── 결제 알림
│   └── 새 리포트 알림
│
└── 👤 마이페이지
    ├── 자녀 관리
    │   ├── 자녀 프로필
    │   └── 프라이빗 리포트 (자녀별)
    ├── 결제 내역 (학원별 / 자녀별)
    ├── 가족 계정 연결 (아빠/조부모)
    └── 설정 (알림, 바우처)
```
