## Persona S1 — "스케줄 관리 지옥의 초등 학부모" 김수진 (38세)

### 기본 정보
- **직업**: 대기업 마케팅 팀장 (주 5일 출근)
- **거주지**: 서울 강남구 역삼동 래미안 아파트
- **자녀**: 초등 4학년 아들 민준, 초등 1학년 딸 수아
- **학원 현황**: 아들(영어·수학·피아노·농구) / 딸(발레·미술)

### 하루 일과 (학원 관련)
```
07:00  아이들 등교 준비
17:00  퇴근 전 카톡 7개 단체방 확인
        → 영어학원 공지 (준비물 변경)
        → 수학학원 셔틀 노선 변경 안내 (스크롤 10분)
17:30  남편에게 전화: "오늘 민준 피아노 픽업 해줄 수 있어?"
18:00  딸 발레 셔틀 도착 시간 확인 (공지 또 찾기)
19:00  귀가 후 수학 학원비 이체 안내 문자 확인 → 입금
```

### 페인 포인트
- "학원 단톡방이 7개야. 어느 방에서 뭘 공지했는지 기억도 못해."
- "셔틀 시간 바뀐 거 카톡에서 찾는 데 10분. 미치겠다."
- "수아 발레 수업 사진이 모르는 사람들 단톡방에 올라오는 게 불안해."
- "아이 둘 시간표 맞추는 게 퍼즐 게임이야. A학원 끝나고 B학원 셔틀 탈 수 있는지 매번 전화해서 확인."

### 에듀픽 사용 시나리오
1. 앱 열면 "오늘 민준: 영어 17:30 → 셔틀 → 수학 19:00 / 수아: 발레 셔틀 17:45"
2. 셔틀 5분 전 자동 알림 → 카카오 알림톡으로 수신 (앱 안 열어도)
3. 딸 발레 사진 → 프라이빗 리포트로만 확인 (단체방 노출 없음)
4. 피아노 학원 추가 검토 → "수학 셔틀 17:55 도착 후 피아노 18:10 가능 여부" 자동 확인

### 인용구
> "카톡 단체방이 진짜 문제예요. 정보가 거기 있는데
> 찾으려면 엄청 스크롤해야 하고, 우리 아이 사진이
> 모르는 사람들한테도 보이는 게 싫거든요.
> 학원 관련된 거 다 한 앱에서 보고 싶어요."

## Persona S2 — "내신 결과로 증명하는 중학생 엄마" 최미경 (43세)

### 기본 정보
- **직업**: 주부
- **거주지**: 경기도 성남시 분당구 정자동
- **자녀**: 중학교 2학년 딸 예진
- **학원 현황**: 국어·영어·수학·과학 (4개 학원)

### 라이프스타일
```
정보 수집: 네이버 카페 '분당 학부모 모임' 매일 확인
의사결정: 맘카페 + 지인 추천 이후 직접 상담
지출 의향: 학원비 월 100~150만 원 (성적 오르면 더 쓸 의향)
```

### 페인 포인트
- "맘카페에서 '이 학원 내신 잘 올려주나요?' 물어봐야 진짜 정보를 얻는데, 눈치가 보여."
- "학원마다 진도 리포트 형식이 달라서 어디서 얼마나 효과 있는지 비교가 안 돼."
- "시험 기간 특강이 갑자기 추가되거나 시간 바뀌는데 문자 못 받으면 놓쳐."
- "광고성 블로그 후기는 믿을 수가 없어. 진짜 다닌 사람 후기가 필요해."

### 에듀픽 사용 시나리오
1. "분당 중등 수학" + "내신 대비 잘함" 인증 리뷰 필터 → 학원 3개 비교
2. 등록 후 → 매주 강사 진도 리포트 수신 (표준화 형식)
3. 시험 기간 특강 일정 → 앱 자동 추가 + 알림
4. 3개월 후 성적 반영 → 타 학원과 효과 비교

### 인용구
> "맘카페 정보가 제일 믿을 만한데, 거기서도 광고인지
> 아닌지 구분이 잘 안 돼요. 영수증 인증된 후기라면
> 진짜라고 믿을 수 있을 것 같아요."

## Persona S3 — "아이가 처음 학원에 다니는 새내기 엄마" 이하은 (33세)

### 기본 정보
- **직업**: 프리랜서 번역가 (재택근무)
- **거주지**: 서울 송파구 잠실동
- **자녀**: 초등 1학년 아들 (첫 학원 등록 예정)
- **현황**: 어떤 학원을 어떻게 알아봐야 하는지 모름

### 페인 포인트
- "첫 학원인데 어디가 좋은지 아는 사람이 없어. 맘카페 가입했는데 어디서 물어봐야 할지도 모르겠어."
- "셔틀이 있는 학원인지, 안전하게 귀가하는지가 제일 걱정."
- "학원비가 얼마인지, 어떻게 결제하는지도 몰라서 일일이 전화해야 해."

### 에듀픽 사용 시나리오
1. "잠실 초등 영어" + "셔틀 있음" + "1학년 반 있음" 필터
2. 학원 상세 → 셔틀 노선 지도 + "우리 아파트 정류장 있음" 확인
3. 다른 1학년 학부모 인증 후기 확인
4. 등록 후 → 셔틀 도착 알림 + 수업 후 활동 사진으로 안심

## Persona S4 — "행정 업무에 지친 학원 원장" 박성호 (41세)

### 기본 정보
- **직업**: 수학 학원 원장 (강사 2명 포함 총 3명 운영)
- **학원**: 서울 강남구 대치동, 수강생 80명
- **현재 도구**: 카카오톡 단체방 8개, 구글 스프레드시트, 무통장 입금

### 학부모 시장에서의 니즈
- "카톡 8개 방에 같은 공지 붙여넣기하는 데 매일 30분."
- "미납자 확인을 스프레드시트로 하는데 실수가 생겨."
- "학부모들이 카톡 말고 체계적인 앱을 원하는 것 같은데 직접 만들 여력이 없어."
- "새 수강생은 블로그 없으면 어떻게 오나 싶어."

### 에듀픽 사용 시나리오
1. 공지 1회 작성 → 전체 학부모 앱 푸시 + 카카오 알림톡 자동 발송
2. 월 결제일 → 미납자 자동 리마인더, 입금 확인 대시보드
3. 에듀픽 학원 프로필 → 인증 리뷰 누적 → 신규 학부모 유입
4. 셔틀 출발 → 원클릭 → 전체 학부모 알림
